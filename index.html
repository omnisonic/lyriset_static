    Automatic font sizing to fit lyrics perfectly on screen

    Swipe left/right on mobile to navigate between songs

    Tap to Scroll

    Overview

    • View a song's lyrics on a single screen with desktop and tablet.
    • Navigate through your setlist easily.
    • Create sets of songs and save the entire collection privately on your computer
    • Best with tablet or laptop as a 'songbook' for practice and performance.

    Smart Auto-Fit

    • Automatic Font Sizing: Calculates optimal font size so lyrics fit perfectly without overflow
    • Longest Line Detection: Measures the longest line to determine column width
    • Screen Optimization: Maximizes font size while ensuring all content fits on screen
    • Desktop & Mobile: Works seamlessly across all device sizes

    Mobile Navigation

    • Side Swipe: Swipe left or right on the lyrics area to go to next/previous song
    • Tap to Scroll: Tap anywhere on the lyrics to start/stop auto-scrolling
    • Manual Scroll: Drag up/down to scroll manually (stops auto-scroll)
    • Auto-Scroll: Smooth continuous scrolling at optimal speed

    Main Features

    • Add Song (): Submit new songs with lyrics from sites like guitaretab.com
    • Sets (Set ▼): Organize songs into sets (1-10) using the Set dropdown
    • Song Selection (Song ▼): Choose songs from your current set
    • Export (): Save your entire collection as a .lyriset file
    • Import (): Load a previously saved .lyriset file
    • Delete (): Remove the currently selected song
    • Edit (): Modify the currently selected song's details
    • Navigation ( ): Move to previous or next song in your set
    • Font Size ( ): Increase, decrease, or reset text size
    • Clean Lyrics (): Toggle between original and chord-free lyrics
    • Auto-Scroll (): Enable/disable automatic scrolling
    • Theme (): Toggle between light and dark mode

    Button Groups & Layout

    • Top Navigation: About, Song, Set, Add buttons for main actions
    • File Management: Export, Import, Delete, Edit grouped together
    • Song Navigation: Previous/Next buttons for quick browsing
    • Display Controls: Font size and auto-scroll controls
    • Visual Controls: Clean lyrics and theme toggle

    Saving & Loading Your Collection

    • Export: Creates a .lyriset file containing all your songs, sets, and font size preferences
    • Import: Loads a previously saved .lyriset file to restore your complete collection
    • File Format: The .lyriset file is a JSON format that stores all your data locally on your device
    • Backup Strategy: Export regularly to create backups. The file can be stored on your device, external drives, or cloud storage
    • Sharing: Share your .lyriset files with other devices or users - perfect for bands or performance groups
    • 100% Private: No data is collected or stored online. Your lyrics collection lives only in your browser's local storage and your exported .lyriset files

    Adding Songs to Your Collection

    • Step 1: Click the button (top navigation) to open the Submit Lyrics form
    • Step 2: Visit a lyrics site like www.guitaretab.com and copy the song lyrics
    • Step 3: Paste the lyrics into the form along with the song title and artist
    • Step 4: Click "Submit" to add the song to your current set
    • Step 5: Use the Export button (in the file management group) to save your collection as a .lyriset file
    • Result: Your songs are now in your setlist and safely saved in your .lyriset file!

    Managing Your Songs

    • Select a Song: Use the Song ▼ dropdown to choose a song from your current set
    • Edit a Song: Select a song, then click the Edit button to modify it
    • Delete a Song: Select a song, then click the Delete button to remove it
    • Switch Sets: Use the Set ▼ dropdown to organize songs into different sets (1-10)

    Navigation & Display Controls

    • Browse Songs: Use Previous and Next buttons to navigate through your set
    • Adjust Text Size: Use to increase, to decrease, or to reset font size
    • Clean View: Click to toggle between original lyrics and chord-free version
    • Auto-Scroll: Click to enable/disable automatic scrolling during performance
    • Theme: Click to switch between light and dark modes

    Tips

    • Export your collection if you make changes so that you have a backup
    • The clean lyrics ( button) feature can help format text for better viewing. If you include chords in the lyrics, the clean lyrics will hide them.
