    About Lyriset

    Overview

    • Lyriset is an app designed to let you view the entire song lyrics on a single screen. You can create your own sets of songs and navigate through them easily.
    • As a performaing artist I use a tablet as my songbook for practice and performance. I wanted a way to view the entire song on a single screen and be able to navigate through the setlist easily. I also wanted to be able to view the lyrics in a clean format. I could not find a app that simply did this so I created my own.

    Main Features

    • Sets: Organize songs into different sets (1-10) using the Set dropdown
    • Navigation: Use ◀ and ▶ buttons to move between songs in the current set
    • Text Size: Adjust text size with A+ and A- buttons
    • Clean Lyrics: Toggle between original and clean lyrics using the ✱ button

    Managing Songs

    • Add Song: Click "Add Song" to manually enter a new song's details
    • Delete: Remove the current song using the Delete button
    • Import/Export: Backup or restore your song collection to your device or computer
    • Set lists are stored in your browser's local storage only temporarily. To save your song data create permanant backups using the Export button.

    Tips

    • Export your collection regularly as a backup
    • The clean lyrics (* button) feature automatically can help format text for better viewing
    • The default songs that are loaded are by Tom lehrer who released them to the public domain. Thanks Tom!
